# Bluetooth Auto Laptop Lock 🔒

A Python-based system that automatically locks your laptop when a trusted Bluetooth device disconnects.

## Features

* Real-time Bluetooth monitoring
* Automatic system lock
* Background execution
* Smart detection (prevents false triggers)

## Tech

* Python
* Bleak
* PyWin32
* AsyncIO

## Run

pip install -r requirements.txt
python autolock.py
